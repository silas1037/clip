管理提醒： (ayazokkon)
该帖由 ayazokkon 从 【同人游戏课堂】 复制到本区.
Another Day Another Memories
Version 1.1 Build 061103

简单易用的通用型GAL Game引擎
（和亚当“Adam”无关）

网址：http://www.fancy-house.com/ADAM/（此网站进一步完善中）
大小：124,839KB
格式：ZIP
内容：程序本体、使用手册、工具、三个演示脚本及其资源

点击这里下载ADAM：http://www.fancy-house.com/ADAM/files/adam_v11_r061103.zip
点击这里下载脚本教程：http://www.fancy-house.com/ADAM/files/script_tutorial_112.rar

另外，第一款用ADAM开发的移植游戏《秋之回忆～从今以后～again简体中文版》已经发布，可以说again就是一个巨大的ADAM演示脚本。


　　什么是ADAM？
　　ADAM，全称Another Day Another Memories，是一个易于使用、快速上手的GAL Game引擎。利用ADAM，你可以很轻松地做出自己的GAL Game。同时，ADAM还是完全免费的，使用ADAM制作游戏不需要支付任何费用。

相对于其他GAL Game引擎，ADAM具有以下优势：
　·详尽的使用手册。游戏制作者可以快速地进入ADAM的世界
　·众多的文件支持。ADAM可以支持大量的图片和音频格式作为资源
　·高效的运行速度。ADAM使用3D硬件进行图像渲染，符合当前潮流，同时做了大量的内部优化，保证程序在实现所有效果的前提拥有较低的CPU占用率和内存占用率
　·简易的使用方法。ADAM将GAL Game的元素大量抽象化，使游戏制作的门槛极大幅度地降低
　·完整的游戏格式。ADAM默认使用一种资源打包方法，将所有资源打包成文件包，不仅可以节省游戏占用的空间，而且还能加快游戏的读取速度。资源打包也符合游戏制作的要求

使用ADAM的系统要求如下：
　·1GHz及其以上等级的CPU
　·GeForce 2MX / Radeon 7200 / XGI Volari V3 / Intel i865G及其以上等级的显卡
　·Windows 98/ME/2000/XP/Vista 操作系统
　·128MB的空闲内存
　·就这些了……
