Darkness-TX commented on 1 Mar

Some games have custom font,it is default_font.dat.
The feature is that there are three font options available
Selecting the third font option in the settings will use default_font.dat.
The first and second font options are system fonts
If it have a custom font, you may change default_font.dat to take effect.If not, you need other method.

fuckfont not used to modify font size,I recommend this method:
Unpack data. pac to get system. ini，put system.ini under the same directory as xxx.exe
edit system.ini:
DEF_FONT_TYPE = 4
to
DEF_FONT_TYPE = 0
(it meens use system fonts,0 meens use the first font options,1 meens use the second font options)
ADD_FONTNAME = "default_font"
to
;ADD_FONTNAME = "default_font"
(it meens disable custom fonts )
than edit FONT_SIZE option,save changes
Finally, delete the save archive or delete save/system.dat,than start the game.
If you don't want to delete the save archive, you can modify save/system.dat.
The offset address of font size for each game may be different.
if FONT_SIZE = 28 in system.ini,
select 1c 00 00 00 in system.dat
If you choose the first font option,it will be 1c 00 00 00 00 00 00 00
If you choose the secont font option,it will be 1c 00 00 00 01 00 00 00
If you choose the third font option,it will be 1c 00 00 00 04 00 00 00
This change occurs when fonts are selected,This offset address may stores the font size.
