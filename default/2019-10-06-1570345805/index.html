游戏提取工具大全

Posted by unknow in 实用软件, 游戏相关 on 2010年2月16日

※消息来源：网络收集

Extraction tools wiki

X-RIPPER （download）
X-Ripper is a file ripper, which means it can find and extract hidden files stored in a big file libraries/archives, mainly used in games. Program can detect and rip various file formats e.g. Audio formats (WAV, WMA, MID, VOC, AU, AIF), Graphics formats (BMP, JPG, PCX, PNG, DDS, GIF, TGA, TIF) and movie formats (AVI, WMV, SMK, BIK). If a program detects any recognizable file format, it automatically saves a file to selected folder. The user can choose which file formats are to be searched.

RPGViewer3.0
支持游戏多，不仅包括日本游戏，欧美、中国，甚至是网游都能提取。而且支持预览，还能提取PNG格式（也就是说能提取出背景透明的立绘）

ExtractData
支持游戏多，不过基本是日本游戏，界面简洁，使用极其方便，只须把想要解开的文件拖到软件界面中去即可

crass
支持游戏多，还支持打包，但界面和操作不太友好，其他工具无法解包的时候可以尝试这个软件。

Susie32
Susie是一个电脑绘图观看程式，由日本的竹村嘉人制作的自由软件。
本程序主要是透过对应的插件，使CG得以变成BMP格式。另外，亦可透过插件，使图像得以存储成为JPEG、TIFF等各种格式。

agth
游戏文本实时提取工具，用途多多，使用方法网上铺天盖地，就不赘述了，此为08年11月版的汉化版

Extractor V2.5
Extractor，一个可以让你从容不迫的应对007,ADAT,APAK,MHW,MIX,MW4,NPAK,PACK,PAK,PBO,PFF,PKR,POD,RES,U,VDF,VPP,WAD,WDIR,WTN,XCR,ZWP等等数十种文件包的解包/封包程序，程序可以在扫描过程中自动按选择的分类识别你需要的文件，有了它,几乎各种游戏的文件包你都可以解开并得到你想要的资源

ＨCG Converter
工口CG Converter能自动并正确判断所支持的封包，能提供一些系统的具体rip选项。对以象dat这样的归档方式的系统，工口CG Converter有不俗的效果。同时它也一些自己的特行，如bmp强制提取，异或解密，alpha反转。

Mami List
著名GALGAME图片浏览解压工具，是专门针对ELF公司的GALGAME制作的， 体积小，效果好，速度快，功能多样，甚至有简单的图片处理功 能。能将图片无损输出为BMP文件。

KID资源提取器
可以提取KID公司旗下大多数游戏,不过经本人测试MO again无法提取

Gameview
和RPGviewer类似,不过支持的游戏有些不同

EgoUnpacker
    支持提取Studio e.go!公司的游戏。

LeafUnpacker
    支持提取Leaf公司的游戏。
    这两款都是windseven的作品，采用类似于WinZip的界面，仅支持解包。不过他在ys168上的网盘已经不存在了。

IGFTool 0.264 (by afe)
    支持提取Illusion公司的游戏

GameSprite 2004 (by 胡泽标)
    http://resource.gameres.com/GameSprite2004.rar
    主要支持欧美的游戏，共享软件。不过很久没有更新了

GameViewer 5.0 (by prefetchnta)
    http://prefetchnta.blog.ccidnet.com/blog-htm-do-showone-uid-20555-type-blog-itemid-218668.html
    主要支持日系的游戏。从这几次的更新来看，作者似乎更侧重于3D模型的查看

asmodean的命令行工具
    http://plaza.rakuten.co.jp/asmodean/

falcnvrt 0.15 2008.11.21
    http://www.geocities.jp/pokan_chan/
    支持提取Falcom公司的游戏。

GameLoader pro (by Mephist)
    http://www.vector.co.jp/soft/dl/win95/art/se247307.html

Grapholic
    http://www.vector.co.jp/soft/win95/art/se039492.html

Westside
    http://www.westside.co.jp/
    http://www.digitalpad.co.jp/~takechin/download.html

Dragon UnPACKer 5.4.0 2008.11.21
    http://sourceforge.net/projects/dragonunpacker
    Delphi编写的解包工具，开放源代码。

Game Archive UnPacker 0.6.0.1 PRO 2009.1.22
    http://forrox.narod.ru/gaup_en.htm
    Total Commander的插件，共享软件。目前支持200多种档案格式的提取，支持近10种档案格式的修改。

MultiEx Commander 4.3.1 2008.10.21
    http://multiex.xentax.com/
    档案提取和修改工具，共享软件。大约支持500种左右的档案格式。网站有专门的wiki和论坛讨论档案格式。

Game Extractor & Game Imager 2.01 2009.1.20
    http://www.watto.org/extract/info.html
    http://sourceforge.net/projects/gameextractor/
    http://sourceforge.net/projects/gameimager/
    Java编写的共享软件，开放源代码。支持1300+种游戏的解包，不过支持的图片格式较少。

FMV-Extractor 1.2
    http://www.necrotech.de/
    免费软件，支持搜索文件中的图像、音频和视频。适合提取未被压缩和加密的数据。
