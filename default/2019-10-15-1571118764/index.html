Oreimo Ex portable: Extract Live 2D Models (and use it !)

Oreimo, Rewriting Needed

I wrote this page to explain how to extract (or rip) 3d models from the game: Ore no Imouto ga Konna ni Kawaii Wake ga Nai Portable. Oreimo Portable for short.

 

Requirements

  • A Windows computer (on Linux you must use Wine)
  • The game in iso format (Google is your friend)
  • UMDGen (A tool that extracts the content of an PSP iso file)
  • OreimoPExtractor (Tool for extract raw files and convert textures files to png)

Extract characters’s sprites

After unpacking the chr.afs file, you a lot of .bip and .t2p files

What can we do with those files ?

  • Here, a member provide a C script for packing and unpacking files with lzss algorithm. http://w11.zetaboards.com/corpseparty/topic/11309987/1/ (Archive)
  • http://www.psdevwiki.com/ps3/GimConv
  • https://warosu.org/jp/thread/9032155
  • https://archive.nyafuu.org/c/thread/1870501/#q1875482
  • https://www.loc.gov/preservation/digital/formats/fdd/fdd000305.shtml

 

How extract 3d models and textures

1-Extract files from the game

2-Extract Live2D data

Launch OreimoP Extractor and seleted RES.DAT. Next extract only the live2d.dat file.

3-Extract Live2D data

Launch OreimoP Extractor and open live2d.dat. Now, extract live2d_model.dat, live2d_motion.dat, live2d_texture.dat.

4-Extract Live2D models

5-Extract Live2D textures

5-Add textures to the model

Textures

Directory structure

When you finaly got your model and it textures files, you have to create some directories. In fact, the live 2d viewer load textures and motions from a specific location.

Legend

  • +: folder
  • -: file
Exemple with Kuroneko
+ My_Working_Folder
  - kuroneko_seifuku.moc
  + kuroneko_seifuku
    - kuroneko_texture_00.png
    - kuroneko_texture_01.png
    - kuroneko_texture_02.png
